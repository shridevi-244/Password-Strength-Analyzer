# Password Strength Analyzer

## Description
A Python-based tool that evaluates the strength of user-entered passwords.

## Features
- Checks password length
- Checks uppercase and lowercase letters
- Checks numbers and special characters
- Detects common passwords
- Classifies passwords as Weak, Medium, or Strong
- Suggests stronger password alternatives

## Technologies Used
- Python

## Sample Input
Password123!

## Sample Output
Password Strength: STRONG
